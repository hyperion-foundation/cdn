# Tentang Hyperion Foundation CDN
Hyperion Foundation CDN merupakan sebuah situs web untuk menyimpan konten-konten lama kami. Hampir semua konten yang ada di situs ini untuk kepentingan **gaming**,
dan ada beberapa konten di situs ini tidak terkait dengan gaming.

Kami membuat situs ini karena kami ingin konten-konten lama kami dapat digunakan atau dimainkan oleh semua orang.
Semua konten yang ada di situs ini harus di unduh terlebih dahulu sebelum dipakai atau dimainkan, dan semua konten yang ada di situs ini adalah **GRATIS** (tanpa biaya).
Jadi, anda dapat menikmati konten-konten yang sebelumnya kami tidak bagikan untuk publik.

# Tech Stack Website
Situs web ini berhasil dibuat dengan menggunakan [Node.js](https://nodejs.org), [Next.js](https://nextjs.org) & [Nextra](https://nextra.site).
