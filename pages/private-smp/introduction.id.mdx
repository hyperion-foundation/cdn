## Pengenalan
Hyperion Private SMP merupakan sebuah Minecraft Survival Multiplayer pribadi di Hyperion Foundation. Yang kami maksud"pribadi", maksud kami adalah hanya member terpilih yang dapat bergabung ke Minecraft SMP pribadi kami.
Kami tidak memilih member secara sembarangan. Server SMP kami dihost di **Minecraft: Java Edition**, bukan **Bedrock Edition**.

SMP kami berjalan dalam "musim" / "season". Musim/Season adalah sebuah panggilan yang dibuat oleh developer Hyperion Private SMP ketika **Mojang Studios** ("Mojang") telah merilis versi terbaru untuk Minecraft: Java Edition.

Contoh:
Mojang telah merilis versi baru Minecraft: Java Edition yaitu versi 1.20. Jadi, kami menyebutnya "Season 1.20".

Tidak setiap musim/season di Hyperion Private SMP memiliki cadangan dunia. Hal ini terjadi dikarenakan ketika kami mencoba mencadangkan dunia, terkadang ukuran file dunia tampak terlalu besar dari yang kami harapkan.
Jadi, kami memutuskan jika ukuran file dunia terlalu besar, kami tidak akan mencadangkannya, tetapi kami akan mencoba yang terbaik untuk mencadangkan dari setiap musim yang kami miliki.
